# CLI Reference

Rudel publishes its command surface under `wp rudel` by default.

You can rename only the root command with `RUDEL_CLI_COMMAND` before Rudel boots.

## Core commands

### `wp rudel create`

Create a sandbox.

```bash
wp rudel create [--name=<name>] [--github=<repo>] [--template=<template>] [--clone-db] [--clone-themes] [--clone-plugins] [--clone-uploads] [--clone-all] [--clone-from=<id>] [--owner=<owner>] [--labels=<labels>] [--purpose=<purpose>] [--protected] [--ttl-days=<days>] [--expires-at=<timestamp>] [--type=<type>]
```

Rudel uses the `subsite` engine. No other engine is supported.

### `wp rudel list`

List sandboxes.

```bash
wp rudel list [--format=<format>]
```

### `wp rudel info`

Show sandbox details.

```bash
wp rudel info <id> [--format=<format>]
```

### `wp rudel update`

Update sandbox metadata and policy.

```bash
wp rudel update <id> [--owner=<owner>] [--labels=<labels>] [--purpose=<purpose>] [--protected] [--unprotected] [--ttl-days=<days>] [--expires-at=<timestamp>] [--clear-expiry]
```

### `wp rudel destroy`

Destroy a sandbox.

```bash
wp rudel destroy <id> [--force]
```

### `wp rudel status`

Read Rudel runtime status.

```bash
wp rudel status
```

### `wp rudel cleanup`

Clean up sandboxes by policy or merged-branch state.

```bash
wp rudel cleanup [--dry-run] [--max-age-days=<days>] [--max-idle-days=<days>] [--merged]
```

### `wp rudel logs`

Read, clear, or follow a sandbox debug log.

```bash
wp rudel logs <id> [--lines=<lines>] [--follow] [--clear]
```

## Recovery points and templates

### `wp rudel snapshot`

Create a sandbox snapshot.

```bash
wp rudel snapshot <id> --name=<name>
```

### `wp rudel restore`

Restore a sandbox snapshot.

```bash
wp rudel restore <id> --snapshot=<name> [--force]
```

### `wp rudel template save`

Save a sandbox as a template.

```bash
wp rudel template save <id> --name=<name> [--description=<description>]
```

### `wp rudel template list`

List saved templates.

```bash
wp rudel template list [--format=<format>]
```

### `wp rudel template delete`

Delete a template.

```bash
wp rudel template delete <name> [--force]
```

## GitHub workflow

### `wp rudel push`

Push sandbox changes to GitHub.

```bash
wp rudel push <id> [--github=<repo>] [--message=<message>] [--dir=<dir>]
```

### `wp rudel pr`

Open a pull request from a sandbox branch.

```bash
wp rudel pr <id> [--github=<repo>] [--title=<title>] [--body=<body>]
```

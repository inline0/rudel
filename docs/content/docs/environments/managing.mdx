# Managing Environments

## List sandboxes

```bash
wp rudel list
wp rudel list --format=json
```

## Inspect a sandbox

```bash
wp rudel info alpha-1234
```

## Update metadata

```bash
wp rudel update alpha-1234 --owner=dennis --labels=qa,client-a --protected
```

Apps use the matching app command:

```bash
wp rudel app update demo-1234 --owner=dennis --labels=production,client-a --protected
```

## Read runtime status

```bash
wp rudel status
```

That shows bootstrap installation, runtime directories, and current counts.

## Cleanup

```bash
wp rudel cleanup --dry-run
wp rudel cleanup --max-age-days=14
wp rudel cleanup --merged
```

## Destroy environments

Destroy a sandbox:

```bash
wp rudel destroy alpha-1234 --force
```

Destroy an app:

```bash
wp rudel app destroy demo-1234 --force
```

Destroying an environment removes its Rudel records, generated directory, and multisite site.
